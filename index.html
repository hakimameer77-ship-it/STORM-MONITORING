StormWatch IoT

An intelligent lightning detection system utilizing the AS3935 Franklin sensor and ESP32 for campus-wide early warnings.

System Architecture

1

Sensing

AS3935 detects electromagnetic pulses from lightning up to 40km away.

2

Processing

ESP32 filters background noise (disturbers) and calculates storm distance.

3

Cloud

Blynk IoT Cloud acts as the bridge, receiving telemetry via secured Wi-Fi.

4

Interface

Web dashboard hosted on GitHub visualizes real-time data and logs events.
