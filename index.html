About Project

The StormWatch project is developed to address the risks of lightning strikes at UTHM Pagoh Campus. By leveraging IoT technology, the system provides real-time atmospheric monitoring, allowing campus authorities to take preemptive safety measures.

Hardware

AS3935 Franklin Sensor integrated with ESP32 for high-precision lightning distance estimation.

Connectivity

Real-time data synchronization via Blynk Cloud using Wi-Fi protocols for instant telemetry.

Frontend

Responsive Web Dashboard featuring Chart.js for data visualization and CSV export for analysis.

System Architecture

01
Detection: AS3935 detects RF emissions from lightning strikes up to 40km away.
02
Processing: ESP32 filters disturbers (noise) and calculates storm proximity.
03
Transmission: Data is pushed to Blynk Cloud Servers via secured Wi-Fi.
04
Alerting: The Web UI provides visual alerts and logs events for safety audits.
