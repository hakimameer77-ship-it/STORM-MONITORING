UTHM Pagoh Project

Lightning Early Warning System

Protecting the campus community through real-time RF atmospheric monitoring and IoT telemetry.

Project Overview

This project addresses lightning strike risks at UTHM Pagoh. Using the AS3935 Franklin sensor, the system calculates the distance of storm heads up to 40km away and provides immediate alerts via this web interface.

Hardware

ESP32 + AS3935 sensor for signal processing.

Cloud

Blynk IoT Cloud for global data synchronization.

Frontend

Responsive JS Dashboard with Chart.js visualization.
